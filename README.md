## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key.
3. Click [here](https://mega.nz/file/hR9wBAQL#LXydkZlDEIjQQIzW3xpnaTziTJay47SzrMT1emjyokU) to see the key.
4. Run the app:
   `npm run dev`
